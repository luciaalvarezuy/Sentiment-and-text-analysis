# Dataset for Sentiment Analysis

This directory contains the dataset used in the notebook:

**`Text_and_sentiment_analysis_of_a_dataframe.ipynb`**

The file **`Tweets.csv`** includes airline-related tweets that are used to perform sentiment analysis and text classification.

In the corresponding notebook, this dataset is used to demonstrate a complete NLP workflow including:

- text preprocessing and normalization  
- TF-IDF feature extraction  
- supervised machine learning models for sentiment classification  
- model evaluation and interpretation of linguistic features  

The dataset serves as the input data for exploring how customer feedback expressed on social media reflects patterns of satisfaction, complaints, and service disruptions in the airline industry.
